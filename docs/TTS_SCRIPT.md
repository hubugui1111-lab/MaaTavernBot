# TTS 配音脚本

## 0. 开场（5秒）
大家好，今天给大家介绍一款开源免费的炉石传说酒馆战旗全自动脚本——MaaTavernBot。

## 1. 环境设置（15秒）
首先，确保你的模拟器分辨率为 1600 乘 900 横屏，DPI 设置为 240。然后开启 ADB 调试。

## 2. 连接设备（10秒）
打开 MaaTavernBot，输入模拟器的 ADB 端口号，点击连接。连接成功后点击开始。

## 3. 自动导航（15秒）
Bot 会自动检测当前画面，从主菜单进入模式选择，选择酒馆战旗，点击开始寻找对局。

## 4. 英雄选择（8秒）
进入英雄选择界面后，Bot 会自动选择第二个英雄并确认。

## 5. 招募操作（20秒）
进入招募阶段，Bot 会优先升级酒馆，然后自动购买随从，把手牌拖到场上打出，还会使用英雄技能。

## 6. 战斗与重开（10秒）
战斗阶段 Bot 会自动等待，结算后自动退出并重新开始下一局。

## 7. 高级功能（10秒）
支持定时运行，设置时间后到点自动启动和停止。关闭窗口可以最小化到系统托盘。

## 8. 结语（5秒）
完全免费开源，GitHub 搜索 MaaTavernBot 即可下载。感谢观看，求个三连！
