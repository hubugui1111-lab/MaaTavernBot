# MaaTavernBot 使用文档

## 环境准备

### 1. 安装 MuMu 模拟器
- 下载地址：https://mumu.163.com
- **必须设置分辨率为 1600×900（横屏）**
- **DPI 设置为 240**
- 开启 ADB 调试（设置 → 其他设置 → ADB 调试）

### 2. 安装炉石传说
- 在模拟器中搜索并安装"炉石传说"

### 3. 下载 MaaTavernBot
- 解压 `MaaTavernBot.zip` 到任意目录

---

## 使用步骤

### 启动
1. 打开 MuMu 模拟器，启动炉石传说
2. 双击 `MaaTavernBot.exe`
3. 输入 ADB 端口号（MuMu 默认 7555，多开实例如 16416）
4. 点击"查找设备"可列出所有设备
5. 点击"连接"
6. 点击"开始"

### 停止
- 点击"停止"按钮，Bot 停止操作但游戏不关闭
- 点击窗口 × 按钮最小化到系统托盘

### 定时运行
- 勾选"启用定时运行"
- 设置开始时间（如 00:00）和结束时间（如 06:00）
- 到时间自动连接、启动游戏、运行，到结束时间自动关闭游戏

### 命令行模式
```
MaaTavernBot.exe --cli --port 16416
```

## 功能说明

| 功能 | 说明 |
|------|------|
| 自动导航 | 主菜单 → 模式选择 → 酒馆战旗 → 寻找对局 |
| 英雄选择 | 自动选第2个英雄 |
| 升级酒馆 | 每回合自动检测并升级 |
| 购买随从 | 自动从酒馆拖拽购买 |
| 打出手牌 | 自动将手牌拖到场上 |
| 英雄技能 | 每回合自动使用 |
| 饰品选择 | 自动选择随机饰品 |
| 自动重开 | 对局结束后自动退出并重开 |
| 卡住自救 | 同画面超过5分钟自动重启游戏 |

## 常见问题

### Q: 点击连接没反应？
A: 检查 ADB 端口是否正确，模拟器是否已开启 ADB 调试。

### Q: 不买随从/不升级？
A: 确认模拟器分辨率为 1600×900（横屏），DPI 240。

### Q: 一直在某个画面不动？
A: Bot 有卡住检测功能，超过5分钟会自动重启游戏。也可以手动停止后重新开始。

### Q: 如何更新？
A: 下载新版本 zip，解压覆盖原目录即可（`.maa` 缓存目录保留不用删）。
