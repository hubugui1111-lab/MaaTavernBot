## 代码规范

### Python 风格
- 遵循 PEP 8
- 类名：PascalCase（`GameBot`, `PhaseDetector`）
- 函数/变量：snake_case（`play_hand_cards`, `gold_count`）
- 常量：UPPER_SNAKE_CASE（`BOARD_MINION_POSITIONS`）
- 缩进：4 空格

### 文件命名
- Python 文件：snake_case（`game_bot.py`）
- JSON 文件：snake_case（`bg_pipeline.json`）
- 图片文件：snake_case（`end_turn_btn.png`）
- 文档文件：kebab-case（`design-spec.md`）

### 注释原则
- 非显而易见的逻辑加注释
- 注释解释"为什么"，而非"做什么"
- 不写冗余注释（变量名自解释时不需要）

### 错误处理
- 不处理不可能发生的场景
- 外部交互（ADB、网络）加 try/except
- 错误记录到 UI 日志

## 操作规范

### 时序要求
| 操作 | 最小间隔 | 说明 |
|------|----------|------|
| 两次点击 | 200ms | 防止游戏忽略连点 |
| 拖拽操作 | 150ms | 拖拽持续时间 |
| 操作后等待 | 300-500ms | 等待游戏响应 |
| 页面检测间隔 | 500ms | rate_limit 控制 |
| 战斗等待 | 2s | 战斗阶段检测间隔可放长 |

### 点击精度
- 目标区域内随机偏移 ±5 像素（防检测）
- 拖拽终点随机偏移 ±10 像素

### 日志级别
| 级别 | 用途 | 示例 |
|------|------|------|
| INFO | 正常操作 | "购买随从，金币 8→5" |
| WARNING | 异常但可恢复 | "未检测到结束回合按钮，重试" |
| ERROR | 需要关注的问题 | "ADB 连接断开，尝试重连" |

## 模板图片规范

### 截图要求
- 分辨率：1080 × 1920（竖屏）
- 格式：PNG
- 裁剪：只包含目标 UI 元素，尽量小
- 命名：描述性英文 + `_btn`/`_icon`/`_bg` 后缀

### 模板匹配阈值
| 场景 | 阈值 | 说明 |
|------|------|------|
| 唯一按钮 | 0.80 | 结束回合、升级等关键按钮 |
| 通用图标 | 0.75 | 金币、设置等小图标 |
| 文字区域 | 0.70 | 排名、阶段文字 |
